# Frontend - Tribunales TFG
Aplicación web React + Vite para asignación automática de tribunales

## Instalación
```bash
npm install
```

## Desarrollo
```bash
npm run dev
```

Abre `http://localhost:5173` en el navegador
La API debe estar corriendo en `http://localhost:8000`

## Build
```bash
npm run build
```

## Características
- ✅ Subida de archivos Excel
- ✅ Procesamiento automático de tribunales
- ✅ Descarga de CSVs por grado
- ✅ Interfaz responsiva y moderna
